Operating systems are software of a system which administers computer hardware, resources, software and gives services common to computer programs.

In time-sharing operatings systems can schedule tasks towards efficient utilization of the system and also can have within accounting software for processor time and cost allocation, storage, printing and similar resources.

The operating system acts as a go between between hardware and computer applications for input and output of hardware functions and also memory allocation, though the application code is usually run directly by the hardware and often makes system calls to or is interupted by OS function. Operating systems are found on many devices which contain a computer like cellular phones, web servers, super computers and video game consoles.
